# 连续数字[[../../数据结构与算法/数学/逆元]]的线性递推

p就是MOD

```CPP
inv[1] = 1;
rep(i, 2, n + 1) inv[i] = (p - inv[p%i]*(p/i) % p); // 从左往右线性递推
```

# 连续阶乘逆元的线性递推

求[[../../数据结构与算法/数学/组合数学]]时比较有用

假设 $n!$ 的逆元是 $b$ 表示的是 $\frac{1}{n!}$ 那么 $n-1$ 阶乘的逆元就是 $\frac{1}{n!}*(n-1)$

```CPP
inv[n] = b 
inv[i] = ((i + 1) * inv[i + 1]) % MOD; // 从右往左线性递推
```
