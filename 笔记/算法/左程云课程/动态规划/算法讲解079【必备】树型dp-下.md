# 好题
## # P2014 [CTSC1997] 选课

[P2014 [CTSC1997] 选课](https://www.luogu.com.cn/problem/P2014)

## 思路

先建图，新增一个0结点把所有点连上

$dp[i][j][k]$ 表示第 $i$ 个结点的前 $j$ 个子树中选了 $k$ 门课的最大学分

每个节点有若干子树，每棵子树都要选择若干结点

状态转移：

第j颗树不选： $dp[i][j][k]=max(dp[i][j][k],dp[i][j-1][k])$ 
选：$dp[i][j][k]=max(dp[i][j][k],dp[i][j-1][k-s]+dp[j对应的树][j的所有子树数量][s])$

```CPP
#define rep(i, j, k) for (int i = (j); i < (k); i++)
#define vi vector<int>

int dp[310][310][310];

int main() {
    IOS;
    int n, m;
    cin >> n >> m;
    m++;

    vector<vi> g(n + 1);
    vi weight(n + 1, 0);
    vi indegree(n + 1, 0); // 入度

    rep(i, 1, n + 1) {
        int u, w;
        cin >> u >> w;
        if (u != 0) {
            g[u].push_back(i);
            indegree[i]++;
        }
        weight[i] = w;
    }
    rep(i, 1, n + 1) if (indegree[i] == 0) g[0].push_back(i);
    rep(i, 0, 310) rep(j, 0, 310) rep(k, 0, 310) dp[i][j][k] = -1;
    function<int(int, int, int)> f = [&](int i, int j, int k) {
        if (k == 0) return 0; // 没有名额
        // 选择i下方除了i的前0颗子树，则只选择i号结点
        // 或者是只有1个名额，不能跨过i选其他名额
        if (j == 0 || k == 1) return weight[i];
        if (dp[i][j][k] != -1) return dp[i][j][k];

        int ans = f(i, j - 1, k);
        int v = g[i][j - 1];
        for (int s = 1; s < k; s++) {
            ans = max(ans, f(i, j - 1, k - s) + f(v, (int)g[v].size(), s));
        }
        dp[i][j][k] = ans;
        return ans;
    };
    f(0, (int)g[0].size(), m);

    cout << dp[0][g[0].size()][m] << endl;
    return 0;
}
```
