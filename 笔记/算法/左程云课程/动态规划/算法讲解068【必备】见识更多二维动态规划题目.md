## 题目1

不同的[[子序列]]
给你两个字符串s和t
统计在s的所有子序列中
有多少个子序列等于t
测试链接 : [https://leetcode.cn/problems/distinct-subsequences/](https://leetcode.cn/problems/distinct-subsequences/)

### 理解

对于s和t，就是选取s的前i个字符串和t的j个字符串进行匹配

对于当前的i和j，首先要确定j肯定是全都要的，那么现在判断i要不要

如果i不要，就是选择s的i-1长度与t的j长度进行匹配

然后判断s的i长度的最后一个和t的j长度的最后一个

如果相等，就是 $dp[i][j]=dp[i-1][j-1]$

```C++
const int N = 1010;
int dp[N][N] = { 0 };
int MOD = 1e9 + 7;
int dp2[N] = { 0 };
class Solution {
public:
    /*
    dp[i][j]表示，s的i长度和t的j长度下的匹配个数
    */
    int numDistinct(string s, string t) {
        //这里的i和j表示长度
        for (int i = 0; i <= s.length(); i++) {
            //i取不管多少个，j取0时，能匹配的长度为1，即s的前i个串能与空集匹配的也只有空集
            dp[i][0] = 1;
            if (i == 0)continue;
            for (int j = 1; j <= t.length(); j++) {
                //对于长度为i和j匹配，第一种，不要当前长度对应的s的最后一个字符s[i-1]
                //要s[0]到s[i-2]的部分与t0到j-1部分的长度
                dp[i][j] = (dp[i - 1][j]) % MOD;
                //如果ij对应的相等，那么 加上
                if (s[i - 1] == t[j - 1]) {
                    dp[i][j] += dp[i - 1][j - 1];
                    dp[i][j] %= MOD;
                }
            }
        }
        return dp[s.length()][t.length()];
    }

    //空间优化
    int numDistinct2(string s, string t) {
        dp2[0] = 1;
        for (int i = 1; i <= s.length(); i++) {
            for (int j = t.length(); j >= 1 ; j--) {
                if (s[i - 1] == t[j - 1]) {
                    dp2[j] += dp2[j - 1];
                    dp2[j] %= MOD;
                }
            }
        }
        return dp2[t.length()];
    }

};
```



## 题目2

编辑距离
给你两个单词 word1 和 word2
请返回将 word1 转换成 word2 所使用的最少代价
你可以对一个单词进行如下三种操作：
插入一个字符，代价a
删除一个字符，代价b
替换一个字符，代价c
测试链接 : [https://leetcode.cn/problems/edit-distance/](https://leetcode.cn/problems/edit-distance/)

注意：
测试里说的题意，只是编辑距离问题的一种情况，请掌握完整的编辑距离问题

对于单词s和t

现在仍旧按最后一个进行分类

插入字符，代价a
删除字符，代价b
替换字符，代价c

- 假定$s[i-1]$参与变换，即存在于最后的结果中
	- 让$s[i-1]$与$t[j-1]$相等
		- 本来就已经相等
			- $dp[i][j]=dp[i-1][j-1]$
		- 本来不相等，需要替换
			- $dp[i][j]=dp[i-1][j-1]+c$
	- 让$s[i-1]$与$t[j-2]$相等
		- 需要增加$t[j-1]$
			- $dp[i][j]=dp[i][j-1]+a$
- $s[i-1]$不参与变换，就是说不在最后的结果中
	- 将$s[i-1]$删除
		- $dp[i][j]=dp[i-1][j]+b$
