# 题目1

## 题意
苹果和盘子
有m个苹果，认为苹果之间无差别，有n个盘子，认为盘子之间无差别
比如5个苹果如果放进3个盘子，那么(1, 3, 1) (1, 1, 3) (3, 1, 1)认为是同一种方法
允许有些盘子是空的，返回有多少种放置方法
测试链接 : https://www.nowcoder.com/practice/bfd8234bb5e84be0b493656e390bdebf

数的划分方法
将整数m分成n份，且每份不能为空，任意两个方案不相同(不考虑顺序)
比如，m=7、n=3，那么(1, 1, 5) (1, 5, 1) (5, 1, 1)认为是同一种方法
返回有多少种不同的划分方法
测试链接 : https://www.luogu.com.cn/problem/P1025

## 思路

1. 没盘子了，但还有苹果，=>0
2. 没苹果了，但还有盘子，=>1
3. 苹果比盘子多
	1. 盘子用光 =>$f(苹果-盘子，盘子)$
	2. 盘子不用管=>$f(苹果，盘子-1)$
## 代码

两题略有不同，一个允许有空的，一个不让，只需要把不让有空的先每个盘子放一个苹果就行
```CPP
#include <iostream>
using namespace std;
//     苹果    盘子
int f(int m,int n){
    if(m==0)return 1;
    if(n==0 && m !=0)return 0;
    if(n>m){
        return f(m,m);
    }else{
        return f(m-n,n)+f(m,n-1);
    }
}
int main() {
    int m,n;
    cin >> m >> n;
    cout<<f(m-n,n);
}
```

# 题目2
## 题意
最好的部署
一共有$n$台机器，编号$1$ ~ $n$，所有机器排成一排
你只能一台一台的部署机器，你可以决定部署的顺序，最终所有机器都要部署
给定三个数组$no[]$、$one[]$、$both[]$
$no[i]$ : 如果i号机器部署时，相邻没有机器部署，此时能获得的收益
$one[i]$ : 如果i号机器部署时，相邻有一台机器部署，此时能获得的收益
$both[i]$ : 如果i号机器部署时，相邻有两台机器部署，此时能获得的收益
第1号机器、第n号机器当然不会有两台相邻的机器
返回部署的最大收益
$1 <= n <= 10^5$   $0 <= no[i]、one[i]、both[i]$
来自真实大厂笔试，对数器验证

## 思路&代码

1. 区间[[../../../数据结构与算法/dp]]

$dp[l][r]$ 表示区间 $[l,r]$ 全都被部署，但是 $l-1$ 和 $r+1$ 没部署

```CPP
rep(i, 1, n + 1) dp[i][i] = no[i];
for (int len = 2; len <= n; len++) {
	for (int l = 1; l <= n - len + 1; l++) {
		int r = l + len - 1;
		int ans = 0;
		for (int i = l; i <= r; i++) {
			int tmp = one[i];
			if (i != l && i != r) tmp = both[i];
			if (i - 1 >= l)  tmp += dp[l][i - 1];
			if (i + 1 <= r) tmp += dp[i + 1][r];
			ans = max(ans, tmp);
		}
		dp[l][r] = ans;
	}
}
cout << dp[1][n];
```

2. 线性

$dp[i][0/1]$ 表示从$[1,i]$区间被部署且 $i+1$ 位置已经被部署或还没被部署

$dp[i][0]$表示1到i已经被部署，此时$i+1$还未被部署，那么i位置部署时要么是$i-1$位置部署了，要么还没部署

则 $dp[i][0]=dp[i-1][0]+one[i]$ 表示 $i$ 还没部署时 $i-1$ 已经部署完毕，同时 $i+1$ 没部署，所以此时贡献为 $one[i]$

类似的还有三种情况，不做赘述

```CPP
dp[1][0] = no[1];
dp[1][1] = one[1];
for (int i = 2; i <= n; i++) {
	dp[i][0] = max(dp[i - 1][0] + one[i], dp[i - 1][1] + no[i]);
	dp[i][1] = max(dp[i - 1][0] + both[i], dp[i - 1][1] + one[i]);
}
cout << dp[n][0];
```

# 题目3

## 题意

求最长[[公共子序列]] 但是经典复杂度为 $O(nm)$ ，nm为两字符串的长度

现在范围： $1 \le n \le 10^6,1 \le m \le 10^3$

## 思路

首先维护一个数组，$pos[i][j]$ 从第 $i$ 位置往后，最近的 $j$ 字符的位置

$dp[i][j]$ 表示长度为 $i$ 的 $s_2$ 的前缀串，想和 $s_1$ 形成长度为 $j$ 的公共子序列，至少需要多长的前缀串才能做到

1. 公共子序列不含 $s_2$ 前缀串的最后字符
	$dp[i-1][j]$
2. 公共子序列含 $s_2$ 前缀串的最后字符
	首先寻找不含最后字符的最长情况，如果找得到这种情况($dp[i-1][j-1]$)
	如果找得到就在此基础上找 $s_2[i]$ 的位置作为ans
	$next[dp[i-1][j-1]][s_2[i]]$
## 代码
```CPP
#define INF 1000000000
void solvedp(string s1, string s2) {
    int n = s1.length(), m = s2.length();
    array<int, 26> a;
    a.fill(INF);
    //构建位置数组
    vector<array<int, 26>> pos(n + 1, a);
    //dp
    vector<vector<int>> dp(m + 1, vector<int>(m + 1, INF));
	
    for (int i = n - 1; i >= 0; i--) {
        for (int j = 0; j < 26; j++) pos[i][j] = pos[i + 1][j];
        pos[i][s1[i] - 'a'] = i;
    }

    for (int i = 0; i <= m; i++) dp[i][0] = 0;

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= i; j++) {
	        // s2[i]不选时，s2的长度为i-1的前缀尝试能不能找得到
            int ans1 = dp[i - 1][j];
            int pre = dp[i - 1][j - 1];
            int ans2 = INF;
            // 想要选择s2[i]，先找s2的如果找得到，基于此寻找对应位置
            if (pre != INF) {
                ans2 = pos[pre][s2[i - 1] - 'a'];
            }
            // 取尽量短的s1的前缀对后续最优，+1是因为ans2表示下标，dp表示的是长度
	        dp[i][j] = min(ans1, ans2 + 1);
        }
    }

    for (int i = m; i >= 1; i--) {
        if (dp[m][i] != INF) {
            cout << "ans = " << i << endl;
            return;
        }
    }
    cout << "ans = " << 0 << endl;
}
```

# 题目4
大楼扔鸡蛋问题
一共有k枚相同的鸡蛋，一共有n层楼
已知一定存在楼层f(0<=f<=n)，从>f的楼层扔鸡蛋一定会碎，从<=f的楼层扔鸡蛋，扔几次都不会碎
鸡蛋一旦碎了就不能再使用，只能选择另外的鸡蛋
现在想确定f的值，返回最少扔几次鸡蛋，可以确保测出该值
1 <= k <= 100
1 <= n <= 10^4
测试链接 : https://leetcode.cn/problems/super-egg-drop/

## 思路

$dp[i][j]$ 表示i个鸡蛋扔j次最多能测出多少楼层

$dp[i][0]=0,dp[1][i]=i$ ，以此为基础，开始推转移

假定$dp[2][6]=a,dp[3][6]=b$ 已知，那么要求 $dp[3][7]$

第一个球用其测试 $a+1$ 层
	1. 如果破了，用两个球测六次正好能测出下方的 $a$ 层
	2. 如果没破，用三个球测六次正好能测出上方 $b$ 层
所以转移显而易见了

##  代码

```CPP
#define LOCAL

#include <bits/stdc++.h>

// CODE

class Solution {
public:
    int superEggDrop(int k, int n) {
        if (k == 1) return n;
        if (k == 2) {
            int l = 0, r = 141;
            while (l < r) {
                int mid = l + r >> 1;
                if (mid * (mid + 1) / 2 >= n)
                    r = mid;
                else
                    l = mid + 1;
            }
            return l;
        }
        vector<vector<ll>> dp(k + 1, vector<ll>(40 + 1));
        for (int i = 0; i <= k; i++) dp[i][0] = 0;
        for (int i = 0; i <= 40; i++) dp[1][i] = i;
        for (int j = 1; j <= 40; j++) {
            for (int i = 1; i <= k; i++) {
                dp[i][j] = dp[i - 1][j - 1] + dp[i][j - 1] + 1;
                if (dp[i][j] >= n) {
                    return j;
                }
            }
        }
        for (int i = 1; i <= 40; i++) {
            cout << dp[2][i] << endl;
        }
        for (int i = 1; i <= 40; i++) {
            if (dp[k][i] >= n) return i;
        }
        return 0;
    }
};
```

# 题目5

## 题意

相邻必选的子序列最大中位数
给定一个长度为n的数组arr
合法子序列定义为，任意相邻的两个数至少要有一个被挑选所组成的子序列
求所有合法子序列中，最大中位数是多少
中位数的定义为上中位数
[1, 2, 3, 4]的上中位数是2
[1, 2, 3, 4, 5]的上中位数是3
2 <=  n <= 10^5
1 <= arr[i] <= 10^9
来自真实大厂笔试，对数器验证

## 思路

拆分成小问
1. 求按这种方式选择的序列的最大累加和

```CPP
int dp[100][2];
int f(vector<int> &a, int n) {
    //dp[i][0]表示位置i的数字选和不选都可以
    //dp[i][1]表示位置i的数字必选
    for (int i = n - 1; i >= 0; i--) {
        dp[i][0] = max(a[i] + dp[i + 1][0], dp[i + 1][1]);
        dp[i][1] = a[i] + dp[i + 1][0];
    }
    return dp[0][0];
}
```

2. 考虑一个x，将所有的数，如果大于等于x的变为1，小于等于x的变为0

那么现在再跑上述的dp，如果答案为正数，说明x可行，那么l=x，反之r=x-1

# 题目6

将珠子放进背包中
给定一个长度为n的数组weights，背包一共有k个
其中weights[i]是第i个珠子的重量
请你按照如下规则将所有的珠子放进k个背包
1，没有背包是空的
2，如果第i个珠子和第j个珠子在同一个背包里，那么i到j所有珠子都要在这个背包里
一个背包如果包含i到j的所有珠子，这个背包的价格是weights[i]+weights[j]
一个珠子分配方案的分数，是所有k个背包的价格之和
请返回所有分配方案中，最大分数与最小分数的差值为多少
1 <= n, k <= 10^5
测试链接 : https://leetcode.cn/problems/put-marbles-in-bags/

## 思路

不是dp，虽然有点像

相当于把数组分成k个子数组，再计算子数组的左右边界

由于必须包括开头和结尾，所以差值中是不包含开头和结尾的

那么假定有 $[a,b,c,d,e,f]$，k=3

任意分成三段：$[a,b],[c,d,e],[f]$ 

可以看成是在bc之间打断点，ef间打断点，同样的，可以在任意两个数之间打断点，再取最大的k-1减去最小的k-1个断点
## 代码

```CPP
class Solution {
public:
    long long putMarbles(vector<int> &weights, int k) {
        int n = weights.size();
        // if(n==k)return 0;
        vector<int> a(n - 1);
        for (int i = 0; i < n - 1; i++) {
            a[i] = weights[i] + weights[i + 1];
        }
        sort(a.begin(), a.end());
        k--;
        long long ans = 0;
        for (int i = 0; i < k; i++) ans -= a[i];
        for (int i = n - 2; i >= n - 2 - k + 1; i--) ans += a[i];
        return ans;
    }
};
```
