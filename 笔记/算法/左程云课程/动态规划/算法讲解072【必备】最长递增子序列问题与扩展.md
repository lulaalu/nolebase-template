# 最长[[递增子序列]]问题

## O($n^2$)的[[../../../数据结构与算法/dp]]

$dp[i]$ 存储以i结尾的最长上升序列的长度

那么每次遍历从该点往前的所有点

```C++
for (int i = 0; i < nums.size(); i++) {
	dp[i] = 1;
	for (int j = 0; j < i; j++) {
		if (nums[j] < nums[i]) {
			dp[i] = max(dp[i], dp[i] + dp[j]);
		}
	}
}
```

## $O(nlog_2n)$的最优解

关键要用一个 $ends$ 辅助数组

$ends[i]$ 表示目前所有长度为 $i+1$ 的递增子序列的最小结尾

由于 $ends$ 是递增的，二分查找进行比较，所以为 $O(nlog_2n)$ 的时间复杂度

```C++
int len=0;//表示ends的长度
for (int i = 0,find; i < nums.size(); i++) {
	find = upper_bound(ends, ends + len, nums[i]) - begin(ends);
	//由于要的是递增子序列，find要的是>=nums[i]的最左位置，防止出现连续相等的情况
	
	while (find - 1 >= 0 && ends[find - 1] == nums[i]) find--;
	if (find == len) {
		ends[len++] = nums[i];
	}
	else {
		ends[find] = nums[i];
	}
}
return len;
```

如果要的是非降序子序列，可以做微调
