# [[矩阵]]快速幂

矩阵相乘要求，a * b，a的行数等于b的列数

$$\left[ 
\begin{matrix}
a_1 & a_2\\
a_3 & a_4 
\end{matrix} 
\right]*
\left[ 
\begin{matrix}
b_1 & b_2\\
b_3 & b_4 
\end{matrix} 
\right]=
\left[ 
\begin{matrix}
a_1b_1+a_2b_3 & a_1b_2+a_3b_4\\
a_3b_1+a_4b_3 & a_3b_2+a_4b_4 
\end{matrix} 
\right]
$$

所以只有"正方形"矩阵能相乘

```CPP
#define vi vector<int>
#define matrix vector<vector<int>>
// 矩阵相乘
matrix multiple(matrix a, matrix b) {
    int n = a.size(), m = b[0].size();
    int s = a[0].size();
    matrix ans = matrix(n, vi(m));
    rep(i, 0, n) {
        rep(j, 0, m) {
            int num = 0;
            rep(k, 0, s) num += a[i][k] * b[k][j];
            ans[i][j] = num;
        }
    }
    return ans;
}

// 矩阵快速幂
matrix power(matrix m, int p) {
    int n = m.size();
    matrix ans = matrix(n, vi(n, 0));
    rep(i, 0, n) ans[i][i] = 1; // 单位矩阵
    for (; p != 0; p >>= 1) {
        if (p & 1) ans = multiple(ans, m);
        m = multiple(m, m);
    }
    return ans;
}

// 打印
void printMatrix(matrix m) {
    for (auto &I : m) {
        for (auto &i : I) cout << i << " ";
        cout << endl;
    }
}
```

# 矩阵快速幂解决一维K阶问题

可解决不带关系依赖的严格递推，如斐波那契数列

## 斐波那契数列

斐波那契数列是
```
下标：0 1 2 3 4 5 6...
	 0 1 1 2 3 5 8...
```

$f(n)=f(n-1)+f(n-2)$ 

有
$$\left[ 
\begin{matrix}
f(1) & f(0)
\end{matrix} 
\right]*
\left[ 
\begin{matrix}
1 & 1\\
1 & 0 
\end{matrix} 
\right]^{n-1}=
\left[ 
\begin{matrix}
f(n) & f(n-1) 
\end{matrix} 
\right]
$$

```CPP
int getf(int i) {
    if (i == 0 || i == 1) return i; 
    matrix f1f0 = {{1, 0}};
    matrix m = {{1, 1}, 
			    {1, 0}};
    return multiple(f1f0, power(m, i - 1))[0][0];
}
```
