# 题目3 格点连线上有几个格点

二维网格中只有x和y的值都为整数的坐标，才叫格点
给定两个格点，A在(x1, y1)，B在(x2, y2)
返回A和B的连线上，包括A和B在内，一共有几个格点
-10^9 <= x1、y1、x2、y2 <= 10^9
测试链接 : [https://lightoj.com/problem/how-many-points](https://lightoj.com/problem/how-many-points)

要想得到A和B之间的点，容易得到 $(x1+n\frac{|x1-x2|}{gcd(|x1-x2|,|y1-y2|}),y1+n\frac{|y1-y2|}{gcd(|x1-x2|,|y1-y2|})$ 也在直线上

不过由于是要在AB之间，所以如果令 $n=gcd$ 即把分母消除，得到的点就是 $(x2,y2)$ 

所以 $n=0$ 到 $n=gcd$ 之间都符合要求，也就是有 $gcd+1$ 个整数点
