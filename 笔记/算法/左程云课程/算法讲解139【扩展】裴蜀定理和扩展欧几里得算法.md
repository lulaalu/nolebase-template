# 裴蜀定理

## 前置相关：辗转相除法正确性的证明

### 证明如下：

要证明辗转相除法就是证明如下关系：
$$gcd(a,b)=gcd(b,a\%b)$$
设 $r=a\%b$

则可得 $a=bq+r \cdots\cdots(1)$  
即 $r=a-bq\cdots\cdots(2)$

设 $u$ 是 $a,b$ 的公因子，$a=xu,b=yu$
带入 $(2)$ 式得：$r=xu-yuq=u(x-yq)$
$\therefore$ $u$ 是 $r$ 的公因子

同理，设 $v$ 是 $b,r$ 的公因子，$b=xv,r=yv$
带入 $(1)$ 式得：$a=xvq+yv=v(xq+y)$
$\therefore$ $v$ 是 $a$ 的公因子

$\because$ $a,b$ 的公因子一定是 $r$ 的因子
	$r$ 的因子一定是 $b,r$ 的公因子
$\therefore$ $a,b$ 的公因子一定是 $b,r$ 的公因子
同理，$b,r$ 的公因子一定是 $a,b$ 的公因子

即 $a,b$ 和 $b,r$ 的公因子集合相互包含，即相等
也就是说$gcd(a,b)=gcd(b,a\%b)$
**证毕**

## 裴蜀定理：
对于一组 $x,y$ 一定存在某一组$a,b$ 使 $ax+by=gcd(x,y)$ 
证明：[[../../数据结构与算法/数学/GCD]]写三四层慢慢推

由此对辗转相除有另外一种理解：
可以认为 $ax+by$ 能到达的最小值就是 $gcd(x,y)$
辗转相除也可以这么理解，取模就是一个减的过程，不断取模，相当于一开始的x和y不断乘倍数再相减

# 扩展欧几里得算法

给出裴蜀定理的最大公约数和一个特解

ax+by=d

```CPP
int d, x, y, px, py;// px和py是下一层的x和y，通过下一层改变当前层最后改变最上层
void exgcd(int a, int b) {
    if (b == 0) {
        d = a;
        x = 1;
        y = 0;
    } else {
        exgcd(b, a % b);
        px = x;
        py = y;
        x = py;
        y = px - py * (a / b);
    }
}
```
