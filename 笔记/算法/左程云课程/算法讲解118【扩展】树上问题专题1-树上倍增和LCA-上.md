# 树上倍增解决[[../../数据结构与算法/图论/树/LCA]]

## 树上寻父

要点：
1. $deep[i]$ 节点i的深度
2. $st[i][p]$ $i$节点往上走 $2^p$ 步到达的节点
	$st[i][0]$ 为 $i$ 的父节点
	$st[i][p]=st[st[i][p-1]][p-1]$
3. 可快速查询从 $i$ 节点往上走到 $s$ 层是什么节点

## 树上倍增

询问 $a$ 和 $b$ 时
1. 先让 $ab$ 跳到同一层
2. 利用倍增算法往上跳，并且跳到的每一层必须是不同的
3. 此时跳到的是离公共祖先最近的两个点，再网上跳一步就是公共祖先了

建立预处理结构 $O(n\log n)$，查询 $O(\log n)$

优势是可以在线查询

## 代码

```CPP
// https://www.luogu.com.cn/problem/P3379
#include <bits/stdc++.h>
using namespace std;

#define int long long
#define endl "\n"
#define IOS ios::sync_with_stdio(0), cin.tie(0), cout.tie(0);
#define rep(i, j, k) for (int i = (j); i < (k); i++)
#define vi vector<int>
const int N = 5e5 + 10;
const int LIMIT = 22;

vi g[N];
int st[N][LIMIT];
int depth[N];

void dfs(int i, int fa, int dep) {
    st[i][0] = fa;
    depth[i] = dep;
    for (int s = 1; s < LIMIT; s++) {
        if (st[i][s - 1] == 0) break;
        st[i][s] = st[st[i][s - 1]][s - 1];
    }

    for (int &u : g[i])
        if (u != fa) dfs(u, i, dep + 1);
} 
int lca(int u, int v) {
    int du = depth[u];
    int dv = depth[v];
    if (du < dv) {
        swap(du, dv);
        swap(u, v);
    }
    // 将深度u、v的深度跳到同一层
    for (int j = LIMIT - 1; j >= 0; j--) {
        if (depth[st[u][j]] >= dv) {
            u = st[u][j];
        }
    }

    // 如果查询的u已经是v的父，会出现相等情况
    if (u == v) return u;

    // 从同一层往上跳
    for (int j = LIMIT - 1; j >= 0; j--) {
        if (st[u][j] != 0 && st[u][j] != st[v][j]) {
            u = st[u][j];
            v = st[v][j];
        }
    }
    return st[u][0];
}
signed main() {
    IOS;
    int n, m, s;
    cin >> n >> m >> s;
    rep(i, 0, n - 1) {
        int u, v;
        cin >> u >> v;
        g[u].push_back(v);
        g[v].push_back(u);
    }

    // dfs将深度 和 st表第一层的信息预处理
    dfs(s, 0, 1);
    depth[0] = -1;

    rep(i, 0, m) {
        int u, v;
        cin >> u >> v;
        cout << lca(u, v) << endl;
    }

    return 0;
}
```

# tarjan算法解决LCA问题

批量且离线的查询

时间：$O(n+m)$ n为树的大小，m为查询的次数

算法过程：

1）处理所有问题，建好每个节点的问题列表，然后遍历树

2）来到当前节点cur，令visited[cur] = true，表示当前节点已经访问

3）遍历cur的所有子树，每棵子树遍历完都和cur节点合并成一个集合，集合设置cur做头节点

4）遍历完所有子树后，处理关于cur节点的每一条查询(cur, x)

   如果发现x已经访问过，cur和x的最低公共祖先 = x所在集合的头节点

   如果发现x没有访问过，那么当前查询先不处理，等到x节点时再去处理查询(x, cur)得到答案
