# 异或空间线性基

## 定义

一堆数字能得到的异或和结果能被这一堆中更少的数字得到，就说更少的这一堆数字式异或线性基

## 理论基础

1. 一堆数字中，任意的a和b，用$a\bigoplus b$ 代替 $a$ 或 $b$ 不影响最后的结果
2. 一堆数字中，任意的a和b，如果有 $a\bigoplus b=0$ 那么可以去除一个
3. 一堆数字能否异或出0，在求出异或空间线性基之后，需要被单独标记


a:1   1       0
b:2   2      0

f(a),f(b)

3

0
## 普通消元代码

zero1表示能否异或出0
```CPP
const int BIT = 50;

vi a, basis1;
int n;

bool insert(int num) {
    for (int i = BIT; i >= 0; i--) {
        if (num >> i == 1) {
            if (basis1[i] == 0) {
                basis1[i] = num;
                return true;
            }
            num ^= basis1[i];
        }
    }
    return false;
}

void compute1() {
    bool zero1 = false;
    for (int &i : a) {
        if (!insert(i)) {
            zero1 = true;
        }
    }
}

signed main() {
    IOS;
    cin >> n;
    a.resize(n);
    for (int &i : a) cin >> i;
    basis1.resize(BIT + 1, 0);
    compute1();
    int ans = 0;
    for (int i = BIT; i >= 0; i--)
        ans = max(ans, ans ^ basis1[i]);
    cout << ans << endl;
    return 0;
}
```
