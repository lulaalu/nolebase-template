# 势能分析

## 题目

范围上开平方并求累加和

给定一个长度为n的数组arr，实现以下两种类型的操作

操作 0 l r : 把 $arr[l..r]$ 范围上的每个数开平方，结果向下取整

操作 1 l r : 查询 $arr[l..r]$ 范围上所有数字的累加和

两种操作一共发生 $m$ 次，数据中有可能 $l>r$，遇到这种情况请交换 $l$ 和 $r$

$1 <= n, m <= 10^5，1 <= arr[i] <= 10^{12}$

测试链接 : [https://www.luogu.com.cn/problem/P4145](https://www.luogu.com.cn/problem/P4145)

## 解析

这种类似的问题不太符合是经典线段树能解决的问题

父范围上的某个信息，不能用O(1)的时间，从子范围的信息加工得到

但是思考本题要求的操作，开方操作，实际上当开方到1时数字再开方无意义

此时可以认为当前数字的势能为0，对于 $10^{12}$ 大约只能开方6次，可以认为势能是6

那么只要区间的总势能是0就不需要再操作

所以不用懒标记直接进行处理，如果区间sum等于区间长度时不操作
