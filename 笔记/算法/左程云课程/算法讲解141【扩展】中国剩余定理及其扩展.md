# 中国剩余定理
## 结论

对于
$x\equiv r_1\pmod{m_1}$ 
$x\equiv r_2\pmod{m_2}$ 
$x\equiv r_3\pmod{m_3}$ 
$\cdots\cdots$

$x=c_1+c_2+\cdots+c_n$
$c_1\%m_1=r_1,c_1\%m_2=0,c_1\%m_3=0,\cdots$
所以需要构造出合适的 $c$ 

$LCM=lcm(m)$ 
$a_i=LCM/m_i$ 
$c_i=r_i*a_i*inv(a_i,m_i)$ 即 $LCM/m_i$ 在 $m_i$ 意义下的逆元

最后的解为 $x=k*LCM+x_0$ 最小解就是 $x_0$

**要求：** 所有 $m$ 互质

## 代码

```CPP
#define i128 __int128_t
#define int long long
int d, x, y, px, py;
void exgcd(int a, int b) {
    if (b == 0) {
        d = a;
        x = 1;
        y = 0;
    } else {
        exgcd(b, a % b);
        px = x;
        py = y;
        x = py;
        y = px - (a / b) * py;
    }
}

// num和mod互质
int getInv(int num, int mod) {
    exgcd(num, mod);
    return (x % mod + mod) % mod;
}

const int N = 15;
int m[N], r[N];
signed main() {
    IOS;
    int n;
    cin >> n;
    rep(i, 1, n + 1) cin >> m[i] >> r[i];
    int LCM = 1;
    // 所有m互质，所以LCM为直接相乘
    rep(i, 1, n + 1) LCM *= m[i];
    int ans = 0;
    rep(i, 1, n + 1) {
        int a = LCM / m[i];
        i128 c = (((i128)r[i] * a) % LCM * getInv(a, m[i])) % LCM; // 模板题卡溢出，用i128或龟速乘
        ans = (ans + c) % LCM;
    }
    cout << ans;
    return 0;
}
```

# 扩展欧几里得小结论

由 $ax+by=d$ 

得 $ax+by=c,(c\%d=0)$ 

有解：$x=x_0\frac{c}{d}+n\frac{b}{d},y=y_0\frac{c}{d}+n\frac{a}{d}$ （$x_0、y_0$ 通过exgcd得到）

结论：x的最小非负解是$(x_0*c/d)\%(b/d)$
