# 流程

1. 判断是否连通：
	- 有向图把边当无向图
	- 无向图就**遍历**一遍，或者**并查集**
2. 判定是否**存在**：
	- 判断欧拉路径（无回路）
		- 有向图：恰好只有一个点的**入度=出度+1**，为终点，恰好只有一个点的**出度=入度+1**，为起点，其余点的出度和入度相等
		- 无向图：**恰好**只有**两个点**的度为**奇数**，其余为偶数
	- 判断欧拉回路
		- 有向图：所有点的**入度等于出度**
		- 无向图：所有点的度为**偶数**
	- **注**：存在欧拉回路（即满足存在欧拉回路的条件），也一定存在欧拉路径
3. 寻找路径：从起点开始遍历所有边，当一个点的边遍历完后，再把该点加入答案**栈**
	- 关于为什么遍历完后入**栈**
	- 首先明确，在第二点已经判断出一定有欧拉路径，那么无论怎么遍历，只要从起点开始，把所有点遍历了，最后一定能遍历所有边
	- 如B->A->B->C->A->D->E，起点为B，假设B先访问A，A可以访问B和D，那么如果访问了D并马上记录，就会断开，不能连成一个欧拉路径，所以要等A访问完所有的边，并记录完访问的点后，再把A入栈，当A入栈后，意味着连C都已经被A->B->C访问了，已经入栈，那么只剩B了。
	- 虽然遍历的点的顺序是不符合欧拉回路，但是由于出度入度的限制，第一个遍历完所有相连边的一定是终点，那终点在栈底，不断往前推，就能获取序列了。
# 例1——[P7771 【模板】欧拉路径](https://www.luogu.com.cn/problem/P7771)

## mean

给定连通（有向看成无向）的有向图

求字典序最小的欧拉路径

## code

```cpp
#define all(x) x.begin(),x.end()
using i64 = long long;
using i32 = signed;

const i64 N = 2e5 + 10;

vector<array<i64, 2>> g[N];
bool vis[N]; // 由于会多次访问同一个点，所以vis记录的是
i64 ind[N];
i64 outd[N];
stack<i64> ans;

void dfs(i64 u)
{
    for (i64 i = 0; i < (i64)g[u].size(); i++) {
        i64 v = g[u][i][0];
        i64 ID_ = g[u][i][1];
        if (vis[ID_])
            continue;
        vis[ID_] = 1;
        dfs(v);
    }
    ans.push(u);
}

signed main()
{
    fill_n(vis, N, 0);
    fill_n(ind, N, 0);
    fill_n(outd, N, 0);
    fill_n(g, N, vector<array<i64, 2>>());

    i64 n, m;
    cin >> n >> m;
    i64 cnt = 0;
    for(i64 i = 0; i < m; i++) {
        i64 u, v;
        cin >> u >> v;
        g[u].push_back({v, cnt++});
        outd[u]++;
        ind[v]++;
    }

    bool ok = 1;
    int S = -1, T = -1;
    for (i64 i = 1; i <= n && ok; i++){
        if (ind[i] != outd[i]){
            if (ind[i] - outd[i] == 1){
                if (T == -1) T = i;
                else{
                    ok = 0;
                    T = -2;
                }
            }
            else if (outd[i] - ind[i] == 1){
                if (S == -1)
                    S = i;
                else {
                    ok = 0;
                    S = -2;
                }
            }
            else ok = 0;
        }
    }
    // 如果有开头和结尾就继续，或者，如果是欧拉回路，就继续
    if (!(S > 0 && T > 0 || ok)) {
        cout << "No" << endl;
        return 0;
    }
    // 字典序要求
    for (i64 i = 1; i <= n; i++) sort(all(g[i]));
    // 同样，如果是欧拉回路，字典序要求，从1开始
    if (S == -1) S = 1;
    dfs(S);
    while (!ans.empty()) {
        cout << ans.top() << ' ';
        ans.pop();
    }
    return 0;
}
```

但是！这个写法有一个点会T，所以要对vis用别的处理

主要是把vis变成del，记录下目前已经访问过的边，相当于把这些边都删掉了

```cpp
vector<i64> g[N];
i64 del[N];
i64 ind[N];
i64 outd[N];
stack<i64> ans;

void dfs(i64 u)
{
    for (i64 i = del[u]; i < sz(g[u]); i = del[u])
    {
        i64 v = g[u][i];
        del[u] = i + 1;
        dfs(v);
    }
    ans.push(u);
}
```
