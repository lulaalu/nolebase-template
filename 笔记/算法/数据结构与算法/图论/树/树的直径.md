# [B4016 树的直径](https://www.luogu.com.cn/problem/B4016)

## 解法1

[[树形dp]]

$d$ 数组表示当前节点的子树的最大深度

注意本题没有边权，所以建图时将 $Weight$ 置为1

树的最大直径一定是某个节点的两条链

那么流程是：
1. 遍历子节点y，也就是获取 $d[y]$ 
2. 更新答案$ans$，因为原本有$dp[x]$的值，现在获取了$d[y]$，相加更新$ans$
3. 更新$d[x]$

```CPP
int ans = 0;
void DP(int x, int f)
{
    vis[x] = 1;
    for (int e = Head[x]; e; e = Next[e])
    {
        int y = To[e];
        if (vis[y] || f == y)
            continue;
        DP(y, x);
        ans = max(ans, d[x] + d[y] + Weight[e]);
        d[x] = max(d[x], d[y] + Weight[e]);
    }
}
```

## 解法2 两次dfs


```CPP
int mx_dep = 0;
int mx_i = -1;
void dfs(int x, int dep, int f)
{
    vis[x] = 1;
    if (dep > mx_dep)
    {
        mx_dep = dep;
        mx_i = x;
    }
    for (int e = Head[x]; e; e = Next[e])
    {
        int y = To[e];
        if (vis[y] || f == y)
            continue;
        dfs(y, dep + Weight[e], x);
    }
}
```
