2. 数字开方向下取整（适用于数字过大但是在 $\_\_int128\_t$ 范围内）
```CPP
int sqr(int n) {
    int x = n;
    int y = (x + 1) / 2;
    while (y < x) {
        x = y;
        y = (x + n / x) / 2;
    }
    return x;
}
```

5. log2：
```CPP
int log2(int n){
	int ans=0;
	while((1<<ans)<=(n>>1))	ans++;
	return ans;
}
```
