# 二维计算几何基础

### 基本公式：

- 正弦定理

$$ \frac{a}{sinA}=\frac{b}{sinB}=\frac{c}{sinC}=2R $$
- 余弦定理

$$a^2=b^2+c^2-2abcosA$$
- 叉乘和点乘
	- 点乘 
		- 公式
			- $a=[a_1,a_2,...],b=[b_1,b_2,...]$
			- $a\cdot b = a_1b_1+a_2b_2+...$ 
		- 几何意义
			- $a\cdot b=|a||b|cos\theta$
	- 叉乘
		- 二维
			- $|\mathbf{a} \times \mathbf{b}| = a_1b_2 - a_2b_1$
			- $|\mathbf{a} \times \mathbf{b}| = 2S = |\mathbf{a}||\mathbf{b}|\sin <a,b>$
		- 三维
			- $\mathbf{a} \times \mathbf{b} = (a_2b_3 - a_3b_2, a_3b_1 - a_1b_3, a_1b_2 - a_2b_1)$

### 1 判断一个点在直线的哪边

$A(x_1, y_1)$和$B(x_2, y_2)$确定，要判断的点为$P(x, y)$。

1. 计算向量$\overrightarrow{AB}$，实际上就是求方向向量 $\overrightarrow{v} = (x_v, x_v)$ 和向量$\overrightarrow{AP} = (x - x_1, y - y_1)$。

2. 计算这两个向量的叉乘：

$$\overrightarrow{v} \times \overrightarrow{AP} = x_v(y - y_1) - y_v(x - x_1)$$

3. 如果叉乘的结果大于0，根据右手定则（如下图），相当于 $\overrightarrow{c}$  指向纸外，则点P在直线AB的上方

### 5 [求任意多边形的面积](https://oi-wiki.org/geometry/2d/#%E6%B1%82%E4%BB%BB%E6%84%8F%E5%A4%9A%E8%BE%B9%E5%BD%A2%E7%9A%84%E5%91%A8%E9%95%BF%E5%92%8C%E9%9D%A2%E7%A7%AF)

- 原理：类似于，将多边形分成若干三角形，根据正弦求面积

- 流程：

多边形顶点按顺时针或逆时针记为 $p_1,p_2,\cdots,p_n$ 

选取任意一个辅助点 $O$ ，记向量 $\mathbf{v_i}=p_i-O$ ，即 $\overrightarrow{Op_i}$ ，那么面积表示为

$$S=\frac{1}{2}\Bigg\lvert\sum_{i=1}^{n}\mathbf{v_i}\times\mathbf{v_{{(i\; mod\; n)}+1}}\Bigg\rvert$$

之所以能够选任意 $O$ 

- 在多边形内部，叉乘不会相互抵消，直接将叉乘变为几何意义的式子就是面积公式

- 假如在多边形外部，叉乘方向不同，在多边形外的部分抵消

所以该公式对于任意多边形都有效


# [[pick定理]]

给定顶点均为整点的简单多边形，面积 $A$，内部格点数 $i$，边上格点数 $b$ 的关系：$A=i+\frac{b}{2}-1$ 

# 二维凸包

凸包就是用皮筋包住所有点的图形

## Andrew 算法

时间复杂度为 $O(nlogn)$ ，$n$ 为点集的大小，复杂度主要是在排序上

### 流程

以横坐标优先进行排序

显然最前和最后的一定是凸包上的点

从最前的点开始求下凸壳，从最前的点逆时针往下，发现总是向左走，那么如果出现向右走的情况就退两步，上凸壳同理

### 代码

```CPP
#include <bits/stdc++.h>
using namespace std;

#define int long long
#define endl "\n"
#define IOS ios::sync_with_stdio(0), cin.tie(0), cout.tie(0);
#define ld long double

const int N = 1e5 + 10;
struct P {
    ld x, y;
} p[N], s[N];// 手写栈

ld dis(P x, P y) { return sqrt((x.x - y.x) * (x.x - y.x) + (x.y - y.y) * (x.y - y.y)); }

// 叉乘
ld cp(P a1, P a2, P b1, P b2) { return (a2.x - a1.x) * (b2.y - b1.y) - (a2.y - a1.y) * (b2.x - b1.x); }

bool cmp(P x, P y) {
    ld tmp = cp(p[1], x, p[1], y);
    if (tmp > 0) return 1;
    if (tmp == 0 && dis(p[1], x) < dis(p[1], y)) return 1;
    return 0;
}

signed main() {
    IOS;
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cin >> p[i].x >> p[i].y;
        if (i != 1 && (p[1].y < p[i].y || (p[1].y == p[i].y && p[1].x < p[i].x))) swap(p[1], p[i]);
    }

    sort(p + 2, p + n + 1, cmp);
    s[1] = p[1];
    int cnt = 1;
    for (int i = 2; i <= n; i++) {
        while (cnt != 1 && cp(s[cnt - 1], s[cnt], s[cnt], p[i]) <= 0) cnt--; // 注意是=，如果在同一条线上，取距离远的
        s[++cnt] = p[i];
    }

    s[cnt + 1] = p[1]; // 闭合图形

    ld ans = 0;

    for (int i = 1; i <= cnt; i++) ans += dis(s[i], s[i + 1]);
    cout << fixed << setprecision(5) << ans;
    return 0;
}
```

# 旋转卡壳

求凸包直径

**叉积的绝对值在数值上等于两个向量形成的三角形的两倍**

在以上求凸包的基础上：

```CPP
int mx = 0;
if (cnt < 3) {
	mx = dis_2(s[1], s[2]);// dis_2为距离的平方
} else {
	int j = 3;
	for (int i = 1; i <= cnt; ++i) {
		while (abs(cp(s[i], s[i + 1], s[i], s[j])) <=
			   abs(cp(s[i], s[i + 1], s[i], s[j % cnt + 1])))
			j = j % cnt + 1;
		mx = max(mx, max(dis_2(s[i + 1], s[j]), dis_2(s[i], s[j])));
	}
}
cout << mx << endl; // 输出距离的平方
```
