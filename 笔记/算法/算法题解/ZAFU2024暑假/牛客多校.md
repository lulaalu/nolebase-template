# 牛客多校1 A. A Bit Common

## 题意

给 $n$ , $m$ ，构造一个序列长度为 $n$ ，其中所有的数都小于 $2^m$ 

要求这个序列满足：存在一个子序列，将所有值 $And$ 操作后的值为 $1$ 

求有多少个这样的序列
## 思路

首先考虑有 $k$ 个数的最后一位都是$1$，其他位至少有一个$0$

那么考虑从这 $k$ 个数里选出 $j$ 个数 $And$ 为 $1$ ，剩下的 $k-j$ 不选，但是会很麻烦

所以思考能不能保证 $k$ 个数里一定有一组数满足要求，因为只要满足要求对答案的贡献就是 $1$

我们知道要 And为1，需要保证除了最低位，其他的 m-1 位至少要有1个0

那么思考，每一位全都为1时肯定不行，每一位只有1个0时肯定可以找出子序列And为0

有更多0时也可以找出And为0的子序列

并且不需要考虑选和不选的问题，因为只要考虑有和没有的问题

那么：

- 任取k个位置的最低位为1--$C_{n}^{k}$
- 这k个位置的其他m-1位每一位都分开考虑，排除全是1的情况--$2^{k}-1$
- 再考虑有m-1位--$(2^n-1)^{m-1}$
- 最低位为1结束，最低位为0类似--$2^{(m-1)(n-k)}$ 

综上 答案是 $\sum_{k=1}^{n}(C_{n}^{k}·(2^k-1)^{m-1}·2^{(m-1)(n-k)})$
## 代码

快速幂加预处理一下组合数
```CPP
#define int long long
#define vi vector<int>
int ksm(int a, int b, int MOD) {
    int res = 1;
    while (b) {
        if (b & 1) res = res * a % MOD;
        a = a * a % MOD;
        b >>= 1;
    }
    return res;
}
signed main() {
    IOS;
    int n, m, MOD;
    cin >> n >> m >> MOD;
    vector<vi> C(n+1, vi(n+1,1));
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j < i; j++) C[i][j] = (C[i - 1][j - 1] + C[i - 1][j]) % MOD;
    }
    int ans = 0;
    for (int k = 1; k <= n; k++) {
        int s1 = (C[n][k] * ksm(ksm(2, k, MOD) - 1, m - 1, MOD))%MOD;
        int s2 = ksm(2, (m - 1) * (n - k), MOD);
        (ans += (s1 * s2) % MOD)%=MOD;
    }
    cout << ans << endl;
    return 0;
}
```
